В Jinja2 есть:
Конструкция 	Назначение 	                                        Пример использования
{% include %} 	вставка другого шаблона в текущий                	{% include 'header.j2' %}
{% block %} 	определение блока, который можно переопределить 	{% block content %}...{% endblock %}
{% import %} 	подключение шаблона как "модуля" с функциями     	{% import 'helpers.j2' as h %}


В Ansible extends работает только с шаблонизатором через template, и не везде стабильно.
Используйте include — безопаснее.
